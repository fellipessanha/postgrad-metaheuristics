# classificação de problemas

- operacionais: problemas muito recorrentes, que precisam ser resolvidos _rápido_
- táticos: problemas de longo prazo, onde o tempo para a solução pode ser _relativamente longo_
- problema online: já estão _pré-resolvidos_, mas precisam ser alterados antes de serem declaradas _soluções finais_
